== RCMS(Rails Content Management System)

Rails로 만든 간단한 게시판입니다.

다음과 같은 기능이 있습니다.

1. Bootstrap을 이용한 Responsive web
2. 폰, 태블릿, 데스크탑 환경에서 사용가능합니다.


== Getting Started

1. 소스를 다운 받고, 압축을 풀어줍니다.
2. 압축을 풀고, 최상위 디렉토리에서 
3. bundle install
4. rake db:migrate
5. rake db:seed
6. thin start
7. 끝.
